# Тесты для Frontend

## Установка зависимостей

```bash
cd frontend
npm install
```

## Запуск тестов

```bash
npm test
```

## Запуск тестов в режиме наблюдения

```bash
npm test -- --watch
```

## Описание тестов

### ThemeContext.test.js

Тесты для функционала темного/светлого режима:

- ✅ Использование светлой темы по умолчанию
- ✅ Переключение темы при нажатии кнопки
- ✅ Сохранение темы в localStorage
- ✅ Загрузка сохраненной темы из localStorage
- ✅ Применение data-theme атрибута к document.documentElement

### SellCar.test.js

Тесты для компонента продажи автомобиля:

- ✅ Отображение формы продажи автомобиля
- ✅ Валидация обязательных полей
- ✅ Успешная отправка формы с валидными данными
- ✅ Валидация года выпуска
- ✅ Валидация цены
- ✅ Очистка формы после успешной отправки
- ✅ Обработка ошибок сервера

## Примечания

- Тесты используют моки для axios
- Все моки очищаются перед каждым тестом
- Используется React Testing Library для тестирования компонентов
